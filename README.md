# Student-GRADER (C++)

##  Description
A console-based C++ application that calculates students' average marks and determines overall pass or fail status based on user input.

This project was developed to strengthen understanding of programming fundamentals such as loops, conditional statements, and user input handling.

##  Features
- Accepts marks for multiple students
- Calculates average marks
- Displays overall pass/fail result
- Simple and interactive command-line interface

## Concepts Used
- Loops
- Conditional Statements
- Variables and Data Types
- User Input & Output
- Basic Arithmetic Operations
